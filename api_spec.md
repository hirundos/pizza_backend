## 👤 로그인/회원가입 (Login)

| 메서드 | URL       | 설명             | 요청 바디                |
|--------|-----------|-----------------|---------------------------|
| POST   | /signup   | 회원가입         | `{ "id": "", "name":"" ,"pw":"" }`    | 
| POST   | /login    | 로그인           | `{ "id": "", "pw:""}`    | 
| GET   | /logout    | 로그아웃         | -                         |

---

## 🛒 주문 (Order)

| 메서드 | URL        | 설명              | 요청 바디                     | 
|--------|------------|------------------|-----------------------------|
| GET    | /myorder   | 내 주문 목록 조회  | -                             | 
| POST   | /order     | 주문 요청         | `[{ "name": "chicken pizza", "quantity": 2, size":"M" } , {"name":"",...}]`| 
| GET    | /pizza     | 피자 종류 조회     | -                             |
